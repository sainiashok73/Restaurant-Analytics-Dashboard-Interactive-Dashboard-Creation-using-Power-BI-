# Restaurant-Analytics-Dashboard-Interactive-Dashboard-Creation-using-Power-BI

## Description
I developed a Restaurant Analytics Dashboard during my internship at Cognifyz Technologies using Power BI to analyze restaurant data and generate business insights. The dashboard visualizes restaurant ratings, cuisines, price ranges, online delivery trends, customer votes, and global restaurant distribution using interactive charts and KPIs.

## Dataset
- <a href="https://github.com/sainiashok73/Restaurant-Analytics-Dashboard-Interactive-Dashboard-Creation-using-Power-BI-/blob/main/Dataset%20.csv">Dataset</a>

## Project Objective

The objective of this project is to analyze restaurant data to understand customer preferences, restaurant ratings, cuisine popularity, pricing trends, and online delivery patterns. The dashboard helps generate business insights and supports better restaurant and customer-focused decision-making.

## Process
Collected restaurant dataset provided during internship at Cognifyz Technologies.
Imported and cleaned the raw dataset in Power BI.
Removed duplicate and missing values for accurate analysis.
Performed data transformation using Power Query.
Created calculated measures and KPIs for analysis.
Designed KPI cards for Total Restaurants, Aggregate Rating, and Votes.
Created visualizations for cuisines, ratings, price ranges, delivery analysis, and restaurant locations.
Added interactive filters and dashboard elements for better user experience.
Built an interactive Power BI dashboard for restaurant analytics and visualization.
Generated insights to understand customer preferences and restaurant performance trends.
